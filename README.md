# test-data-service

Веб-сервис для предоставления тестовых данных


### Запуск сервиса

1. Создайте виртуальное окружение

```
python3 -m venv venv
```

2. Активировать виртуальное окружение: 

```
source venv/bin/activate
```

3. Установить зависимости: 

```
python setup.py install
```

4. Запустите сервер
```
test-data-service
```
